# 🚀 SpeakAid – Future Roadmap & AI Integration

This document outlines features that transform SpeakAid from a basic routine app into a meaningful, privacy-respecting assistive tool for neurodivergent users.

---

# 🧠 1. Privacy-First AI & ML Features

Instead of using invasive camera/microphone tracking, SpeakAid focuses on **Cognitive & Executive Function** support using on-device AI.

### A. AI Task Decomposer (Micro-Steps)
- **Goal:** Solve "Big Task Paralysis."
- **Feature:** Caregivers enter a broad task (e.g., "Get Ready for Bed"), and the AI automatically suggests a breakdown of 5-7 micro-steps.
- **Value:** Reduces cognitive load by handling the planning phase for the user.

### B. Adaptive Pacing (ML Timers)
- **Goal:** Reduce "Transition Anxiety."
- **Feature:** An ML model learns the user's natural speed for specific tasks. 
- **Value:** If a user takes longer to "Brush Teeth" than "Wash Face," the app dynamically adjusts the countdown timers so the user never feels rushed or pressured.

### C. Plain Language Simplifier (NLP)
- **Goal:** Increase comprehension.
- **Feature:** Uses Natural Language Processing to convert complex sentences into "Plain Language" (short, direct, action-oriented instructions).
- **Value:** Ensures users with language processing challenges can understand the "What" and "How" instantly.

### D. Semantic "Mood" Discovery
- **Goal:** Support emotional regulation.
- **Feature:** Users type how they feel (e.g., "My brain is noisy"), and the app uses semantic search to suggest the most appropriate routine (e.g., "Sensory Break").
- **Value:** Bridges the gap between feeling an emotion and finding a coping strategy.

---

# 🎯 2. Behavioral Support Features

### “Why This Step Matters”
- **Goal:** Build intrinsic motivation.
- **Feature:** Every step includes an optional "Why" card (e.g., "Brushing teeth keeps your smile bright").
- **Value:** Neurodivergent users often cooperate better when the *logic* behind a demand is clear.

### Reward Customization
- **Goal:** Personalized reinforcement.
- **Feature:** Allow caregivers to swap the "Confetti" for other rewards (e.g., a short clip of a favorite sound, a specific color pulse, or a digital sticker).
- **Value:** Increases engagement by aligning with the user's specific "Special Interests."

---

# 🛠️ 3. Caregiver Analytics (Offline)

### Completion Heatmaps
- **Goal:** Identify "Friction Points."
- **Feature:** A dashboard showing which routines are frequently abandoned or where the user gets "stuck."
- **Value:** Helps parents identify if a specific step is too difficult or sensory-overwhelming.
